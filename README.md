# The todo app

Monorepo example

Required node 18+
Steps to run

1. npm install -g pnpm
2. In the repo do pnpm install
3. After install do pnpm dev
4. Both apps will start in 3000 and 3001 respsectively
5. For unit test go to apps/todo-api
6. pnpm test.

7. Apps
   /web-app
   /api
   2.Packages
   /conifg
   /lint
   /ui modules
